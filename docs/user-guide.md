## Block Usage

### Adding a Block

1. Edit a page/post
2. Click **+** to add block
3. Search "Wbcom" or browse "Wbcom Essential" category
4. Click to insert

### Block Settings

Each block has settings in:

1. **Toolbar** — Quick formatting options
2. **Sidebar** — Full settings panel (Inspector Controls)

### Common Controls

| Control | Description |
|---------|-------------|
| Typography | Font size, family, weight |
| Colors | Text, background, accent |
| Spacing | Margin, padding |
| Border | Radius, width, color |
| Alignment | Left, center, right, full |

## Tips & Best Practices

### Performance

1. **Lazy load carousels** — Use "Load on scroll" where available
2. **Limit posts per query** — Keep under 12 for carousels
3. **Optimize images** — Use appropriate sizes

### Accessibility

1. **Use heading hierarchy** — H1 → H2 → H3
2. **Add alt text** — For all images
3. **Test keyboard navigation** — Especially for accordions/tabs

### Mobile Responsiveness

1. **Preview on mobile** — Use Elementor/Gutenberg preview
2. **Adjust columns** — Reduce for smaller screens
3. **Test touch interactions** — Carousels, dropdowns
