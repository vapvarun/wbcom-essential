# Elementor Widgets Guide

WBcom Essential provides **43 Elementor widgets** organized into 3 categories. This guide covers each widget with its purpose, settings, and practical use cases.

---

## Widget Categories Overview

| Category | Widgets | Purpose |
|----------|---------|---------|
| General | 27 | Sliders, carousels, content layouts, forms |
| BuddyPress | 11 | Member grids, groups, forums, activity |
| WooCommerce | 5 | Products, reviews, banners |

---

## How to Use Widgets

### Adding Widgets

1. Edit any page with Elementor
2. Open the widget panel (left sidebar)
3. Search for "Starter Templates" or scroll to find the category
4. Drag any widget onto your page
5. Configure settings in the left panel

### Widget Settings Panels

Every widget has three tabs:

| Tab | Contains |
|-----|----------|
| **Content** | Main content settings (text, items, queries) |
| **Style** | Visual customization (colors, typography, spacing) |
| **Advanced** | Positioning, motion effects, responsive controls |

## Tips for All Widgets

### Performance

1. **Limit carousel items** to 6-8 for fast loading
2. **Optimize images** before uploading
3. **Use lazy loading** where available

### Consistency

1. **Use Global Colors** in Elementor for consistency
2. **Create widget templates** for reuse
3. **Save sections to library** for quick access

### Mobile

1. **Preview on mobile** before publishing
2. **Adjust responsive settings** per widget
3. **Test touch interactions** for carousels

### Accessibility

1. **Use proper heading hierarchy** (H1 → H2 → H3)
2. **Add alt text** to all images
3. **Ensure color contrast** meets WCAG standards
