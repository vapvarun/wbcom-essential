**Version:** 4.0.2

**License:** GPL v2 or later

## Overview

Wbcom Essential is a comprehensive companion plugin for WordPress themes, providing **43 Elementor widgets** and **38 Gutenberg blocks** for BuddyPress, WooCommerce, and general WordPress functionality.

## Quick Start

1. Upload `wbcom-essential` to `/wp-content/plugins/`
2. Activate via **Plugins** menu
3. For Elementor: Widgets appear under "Wbcom Essential" category
4. For Gutenberg: Blocks appear under "Wbcom Essential" category
