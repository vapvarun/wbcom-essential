# Gutenberg Blocks Guide

WBcom Essential provides **45 Gutenberg blocks** organized into 7 categories. This guide covers each block with its purpose, settings, and use cases.

## Block Categories Overview

| Category | Blocks | Purpose |
|----------|--------|---------|
| Starter Pack - Header | 4 | Site branding and navigation |
| Starter Pack - Design | 14 | Visual and decorative elements |
| Starter Pack - Content | 8 | Text and interactive content |
| Starter Pack - Blog | 8 | Post displays and queries |
| Starter Pack - Marketing | 4 | Conversion-focused elements |
| Starter Pack - BuddyPress | 11 | Community features |
| Starter Pack - WooCommerce | 2 | E-commerce features |

---

## Header Blocks

### Branding

Display your site logo with title and tagline options.

**Settings:**
- Display mode: Logo only, Title only, or Both
- Logo source: Site logo or custom image
- Title tag: H1-H6
- Link options: Home or custom URL

**Use cases:**
- Site header
- Footer branding
- Email-style letterhead sections

---

### Site Logo

Simple logo block with size and link controls.

**Settings:**
- Logo image selection
- Width and height controls
- Link to homepage option
- Alignment: left, center, right

**Use cases:**
- Header logo
- Partner/sponsor logos
- Footer branding

---

### Header Bar

Navigation bar with user account elements.

**Settings:**
- Show/hide search
- Show/hide notifications (requires BuddyPress)
- Show/hide messages (requires BuddyPress)
- Show/hide cart (requires WooCommerce)
- User menu items

**Use cases:**
- Main site header
- Member area navigation
- Community site headers

---

### Smart Menu

Advanced navigation menu with mega menu support.

**Settings:**
- Menu selection
- Layout: horizontal or vertical
- Dropdown animation style
- Mobile breakpoint
- Hamburger icon style

**Use cases:**
- Main navigation
- Footer navigation
- Sidebar navigation

---

## Common Settings

All blocks share these settings:

### Color Settings

- **Use Theme Colors**: Toggle to inherit theme colors
- **Background Color**: Block background
- **Text Color**: Content text color

### Spacing

- **Margin**: Space outside the block
- **Padding**: Space inside the block

### Typography

- **Font Size**: Text size
- **Font Weight**: Bold, normal, light
- **Line Height**: Space between lines

### Advanced

- **Additional CSS Classes**: Custom class names
- **HTML Anchor**: ID for linking
