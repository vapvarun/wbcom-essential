# Blog Blocks

6 blocks for displaying your WordPress posts in beautiful layouts - carousels, sliders, tickers, and timelines.

---

## Post Carousel

Blog posts in horizontal carousel format.

<!-- SCREENSHOT: Post Carousel in editor -->
<!-- SCREENSHOT: Post Carousel on frontend -->

### What You Can Do

- Display posts in sliding carousel
- Filter by category or tag
- Show excerpt, date, author, categories
- Control how many posts appear at once

### Settings

| Setting | Options | Recommended |
|---------|---------|-------------|
| Posts Per View | 1-6 | 3 for desktop |
| Category Filter | Select categories | Your main category |
| Order By | Date, Title, Random | Date |
| Show Excerpt | Yes/No | Yes |
| Excerpt Length | Words | 20 |
| Show Date | Yes/No | Yes |
| Show Author | Yes/No | Optional |
| Autoplay | Yes/No | Yes |

### Best For

- Featured posts on homepage
- Related articles at end of posts
- Latest news section
- Blog highlights

## Posts Carousel

Alternative post carousel with card layout.

<!-- SCREENSHOT: Posts Carousel on frontend -->

### What You Can Do

- Display posts as styled cards
- Multiple cards per slide
- Control image aspect ratio
- Style card appearance

### Settings

| Setting | Options |
|---------|---------|
| Query | Category, Tag, Author |
| Cards Per Slide | 1-4 |
| Show Image | Yes/No |
| Image Ratio | 16:9, 4:3, Square |
| Card Style | Shadow, Border, Flat |

### Best For

- News sections
- Article highlights
- Content recommendations

## Post Slider

Full-width hero slider featuring posts.

<!-- SCREENSHOT: Post Slider on frontend -->

### What You Can Do

- Create dramatic full-width sliders
- Overlay text on featured images
- Control content position
- Add gradient overlays

### Settings

| Setting | Options |
|---------|---------|
| Posts | Select or query |
| Slide Height | Pixels or viewport |
| Overlay Opacity | 0-100% |
| Content Position | Left, Center, Right |
| Show Title | Yes/No |
| Show Excerpt | Yes/No |
| Read More Button | Yes/No |

### Best For

- Hero sections
- Featured article sliders
- News highlights
- Magazine-style homepages

## Posts Revolution

Dynamic posts grid with filtering animation.

<!-- SCREENSHOT: Posts Revolution with filters -->

### What You Can Do

- Display posts with category filters
- Animated filtering transitions
- Load more functionality
- Multiple layout styles

### Settings

| Setting | Options |
|---------|---------|
| Layout Style | Grid, Masonry |
| Filter Tabs | Yes/No (by category) |
| Animation | Fade, Slide, Scale |

### Best For

- Blog archives
- News sections

- Magazine layouts

## Posts Ticker

Scrolling news ticker bar.

<!-- SCREENSHOT: Posts Ticker on frontend -->

### What You Can Do

- Create scrolling headline ticker
- Add label (e.g., "Breaking News")
- Control scroll speed
- Pause on hover

### Settings

| Setting | Options | Recommended |
|---------|---------|-------------|
| Label | Text | "Latest News" |
| Scroll Speed | Slow, Medium, Fast | Medium |
| Direction | Left, Right | Left |
| Post Count | Number | 10 |
| Category | Filter | News category |
| Pause on Hover | Yes/No | Yes |

### Best For

- Breaking news
- Announcements bar
- Updates strip
- Alert notifications

## Choosing the Right Blog Block

| You Want... | Use This Block |
|-------------|----------------|
| Hero section with posts | **Post Slider** |
| Posts in sidebar or homepage section | **Post Carousel** |
| Filterable blog archive | **Posts Revolution** |
| Breaking news strip | **Posts Ticker** |
| Historical blog archive | **Post Timeline** |
| Card-style post display | **Posts Carousel** |
