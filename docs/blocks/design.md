# Design Blocks

12 blocks for creating visually stunning pages. Add animations, sliders, icons, and decorative elements.

---

## Counter

Animated numbers that count up when visitors scroll to them.

<!-- SCREENSHOT: Counter block in editor -->
<!-- SCREENSHOT: Counter animation on frontend -->

### What You Can Do

- Display statistics with animated counting
- Add prefix ($) and suffix (%, +, etc.)
- Control animation speed
- Show decimal numbers

### Settings

| Setting | Options | Example |
|---------|---------|---------|
| Start Number | Any number | 0 |
| End Number | Any number | 500 |
| Prefix | Text | $ |
| Suffix | Text | + |
| Duration | Milliseconds | 2000 |
| Decimal Places | 0-3 | 0 |

### Best For

- "500+ Happy Customers" statistics
- Years of experience
- Projects completed
- Revenue figures

## Countdown

Timer counting down to a specific date.

<!-- SCREENSHOT: Countdown block in editor -->
<!-- SCREENSHOT: Countdown on frontend -->

### What You Can Do

- Count down to any future date
- Show days, hours, minutes, seconds
- Display message when timer ends
- Style inline or stacked layout

### Settings

| Setting | Options | Recommended |
|---------|---------|-------------|
| Target Date | Date picker | Your event date |
| Show Days | Yes/No | Yes for events > 1 day |
| Show Seconds | Yes/No | Yes for urgency |
| Expiry Message | Text | "Sale has ended!" |
| Layout | Inline, Stacked | Stacked for impact |

### Best For

- Product launches
- Sale endings
- Event countdowns
- Webinar registration urgency
