## Pricing Table

Display pricing plans for your products or services.

<!-- SCREENSHOT: Pricing Table in editor -->
<!-- SCREENSHOT: Pricing Table on frontend -->

### What You Can Do

- Create professional pricing columns
- Show original price for sales
- List features with check/cross icons
- Highlight "Most Popular" plan
- Add ribbon badges

### Settings

| Setting | Options | Recommended |
|---------|---------|-------------|
| Title | Plan name | "Basic", "Pro", "Enterprise" |
| Description | Short text | Plan summary |
| Price | Amount | Your price |
| Price Prefix | Text | "$" |
| Price Suffix | Text | "/month" |
| Original Price | Amount | For showing discounts |
| Features | List items | 5-7 key features |
| Feature Icons | Check/Cross | Check for included |
| Button Text | Text | "Get Started" |
| Featured | Yes/No | Yes for middle plan |
| Ribbon | Text | "Best Value" |

### Best For

- SaaS pricing pages
- Membership plans
- Service packages
- Subscription options

### Pro Tips

1. **Show 3 plans** - Basic, Pro, Enterprise works best
2. **Highlight the middle option** - Mark as "Most Popular"
3. **List 5-7 features** - Not too few, not overwhelming
4. **Clear CTA buttons** - Make it obvious how to buy

## Testimonial Carousel

Multiple testimonials in sliding format.

<!-- SCREENSHOT: Testimonial Carousel on frontend -->

### What You Can Do

- Add unlimited testimonials
- Show multiple at once
- Auto-play through testimonials
- Choose card or minimal style

### Settings

| Setting | Options | Recommended |
|---------|---------|-------------|
| Testimonials | Add multiple | 5-6 |
| Layout | Cards, Minimal | Cards |
| Per View | 1-3 | 1-2 |
| Show Ratings | Yes/No | Yes |
| Autoplay | Yes/No | Yes |
| Loop | Yes/No | Yes |

### Best For

- Dedicated testimonials section
- Social proof on landing pages
- Client reviews collection

### Pro Tips

1. **Include photos** - Faces increase trust
2. **Add star ratings** - Visual credibility
3. **Keep it moving** - Enable autoplay
4. **5-6 is optimal** - Not too few, not overwhelming

## CTA Box

Call-to-action box to drive conversions.

<!-- SCREENSHOT: CTA Box variations -->

### What You Can Do

- Create compelling CTA sections
- Combine heading, description, button
- Use as horizontal banner or stacked
- Style with gradients and colors

### Settings

| Setting | Options |
|---------|---------|
| Heading | Your CTA headline |
| Description | Supporting text |
| Button Text | Action text |
| Button Link | URL |
| Secondary Button | Optional |
| Layout | Horizontal, Stacked |
| Background | Color, Gradient, Image |

### Best For

- Newsletter signups
- Free trial offers
- "Get Started" sections
- Contact prompts

### Effective CTA Examples

| Heading | Button |
|---------|--------|
| "Ready to grow your community?" | "Start Free Trial" |
| "Join 10,000+ happy members" | "Sign Up Now" |
| "Questions? We're here to help" | "Contact Us" |
| "Don't miss our updates" | "Subscribe" |

2. **Features** showing benefits
3. **Testimonials** for social proof
4. **Pricing** with clear options
5. **CTA** to close the deal
