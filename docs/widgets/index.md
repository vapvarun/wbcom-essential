## How to Find Widgets

1. Edit any page with Elementor
2. Look in the left panel for widgets
3. Search "Starter Templates" or scroll to find our widgets
4. Drag any widget onto your page

## Widget Settings

Every widget has 3 tabs:

| Tab | What It Controls |
|-----|------------------|
| **Content** | Text, items, what to show |
| **Style** | Colors, fonts, spacing |
| **Advanced** | Position, animations, responsive |

<!-- SCREENSHOT: Widget settings tabs -->

---

## Quick Tips

1. **Use Style tab** - Most visual changes are here
2. **Check Advanced tab** - For responsive hiding/showing
3. **Right-click to duplicate** - Faster than recreating
4. **Save as Global Widget** - Reuse across pages
