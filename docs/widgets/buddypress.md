# BuddyPress Widgets

11 widgets for building community features. **Requires BuddyPress plugin.**

> These widgets only appear in Elementor when BuddyPress is installed and activated.

---

## Members Carousel

Display community members in sliding carousel.

<!-- SCREENSHOT: Members Carousel widget -->

**Key Settings:**
- Count: Number of members
- Type: Active, Popular, Newest, All
- Per View: How many visible
- Show: Avatar, name, meta
- Autoplay: With duration

**Best For:** Homepage member showcase, community highlights

---

## Members Grid

Display members in organized grid layout.

<!-- SCREENSHOT: Members Grid widget -->

**Key Settings:**
- Columns: 2-6
- Per Page: With pagination
- Type: Filter members
- Search: Enable/disable
- Filters: Show filter options

**Best For:** Member directory pages, full listings

---

## Members Lists

Compact member list display.

<!-- SCREENSHOT: Members Lists widget -->

**Key Settings:**
- Count: Number to show
- Type: Active, Popular, etc.
- Avatar Size: Small or Medium
- Show Role: Yes/No

**Best For:** Sidebar widgets, compact displays

## Building a Community Page

### Recommended Widget Combination

```
[Dashboard Intro] - Greet your members

[Members Carousel] - "Our Active Community"

[Groups Grid] - "Explore Groups"

[Forums Activity] - "Latest Discussions"

[Testimonial Carousel] - Member success stories

[CTA] - "Join Our Community"
```
