## Technical SEO

### Schema Markup Suggestions

**Product Schema (for plugin page):**
```json
{
  "@type": "SoftwareApplication",
  "name": "WBcom Essential",
  "applicationCategory": "WordPress Plugin",
  "offers": {
    "@type": "Offer",
    "price": "49",
    "priceCurrency": "USD"
  }
}
```

**FAQ Schema (for FAQ page):**
Use structured data for common questions about blocks, compatibility, etc.

### URL Structure

Recommended URL patterns:
- `/blocks/` - All blocks overview
- `/blocks/members-grid/` - Individual block pages
- `/widgets/` - All widgets overview
- `/integrations/buddypress/` - Integration docs
- `/tutorials/` - How-to content

### Internal Linking Strategy

1. **Feature pages** link to related blocks
2. **Block pages** link to tutorials using that block
3. **Tutorials** link to block documentation
4. **All pages** link to BuddyPress integration if relevant
