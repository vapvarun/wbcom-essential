# Installation & Setup Video Script - WBcom Essential

**Duration:** 3-4 minutes
**Purpose:** Step-by-step installation tutorial
**Tone:** Helpful, clear, patient

---

## Video Details

| Element | Specification |
|---------|--------------|
| Length | 3:00 - 4:00 |
| Style | Screen recording with voiceover |
| Music | Minimal or none |
| Resolution | 1080p minimum |
| Thumbnail | WordPress plugins page with "Install Guide" text |

---

## Script

### INTRO (0:00 - 0:15)

**VISUAL:** WBcom Essential logo

**NARRATOR:**
"In this video, I'll show you how to install WBcom Essential and use your first block. Takes about 5 minutes."

**ON-SCREEN TEXT:** "WBcom Essential - Installation Guide"

---

### DOWNLOAD (0:15 - 0:45)

**VISUAL:** Navigate to download page

**NARRATOR:**
"First, download the plugin. Go to wbcomdesigns.com/downloads/wbcom-essential."

**VISUAL:** Click download button

**NARRATOR:**
"Click download. You'll get a ZIP file—don't extract it. WordPress needs the ZIP file for installation."

**ON-SCREEN TEXT:** "Keep it as a ZIP file"

---

### INSTALLATION (0:45 - 1:30)

**VISUAL:** WordPress admin dashboard

**NARRATOR:**
"In your WordPress admin, go to Plugins, then Add New."

**VISUAL:** Click Upload Plugin

**NARRATOR:**
"Click 'Upload Plugin' at the top."

**VISUAL:** Choose file dialog

**NARRATOR:**
"Choose the ZIP file you downloaded and click 'Install Now'."

**VISUAL:** Installation progress, then Activate button

**NARRATOR:**
"Once it's installed, click 'Activate'. That's it—the plugin is ready to use."

**ON-SCREEN TEXT:** "Plugin activated"

---

### FINDING BLOCKS (1:30 - 2:15)

**VISUAL:** Create new page, open block inserter

**NARRATOR:**
"Let's find the blocks. Create a new page or edit an existing one."

**VISUAL:** Click + button

**NARRATOR:**
"Click the plus button to open the block inserter. You can search for 'Starter Pack' to see all WBcom Essential blocks."

**VISUAL:** Show block categories

**NARRATOR:**
"Or scroll down to find the categories: Starter Pack Header, Design, Content, Blog, Marketing. If BuddyPress is active, you'll also see Starter Pack BuddyPress."

**ON-SCREEN TEXT:** "Search 'Starter Pack' or browse categories"

---

### FIRST BLOCK DEMO (2:15 - 3:15)

**VISUAL:** Add Testimonial block

**NARRATOR:**
"Let's add a block. I'll use the Testimonial block as an example."

**VISUAL:** Click on Testimonial block

**NARRATOR:**
"Search 'Testimonial' and click to add it."

**VISUAL:** Block appears on page

**NARRATOR:**
"The block appears with placeholder content. Click to select it, and you'll see settings on the right side."

**VISUAL:** Show sidebar settings

**NARRATOR:**
"Here you can add the testimonial text, author name, their role, and a photo."

**VISUAL:** Enable Theme Colors

**NARRATOR:**
"Most importantly, find 'Use Theme Colors' and toggle it on. Now the block matches your theme automatically."

**VISUAL:** Preview the page

**NARRATOR:**
"Preview the page and you'll see a professional testimonial that fits your site design."

**ON-SCREEN TEXT:** "Theme Colors - automatic matching"

---

### ELEMENTOR WIDGETS (3:15 - 3:45)

**VISUAL:** Open page with Elementor

**NARRATOR:**
"If you prefer Elementor, the same features are available as widgets."

**VISUAL:** Elementor editor widget panel

**NARRATOR:**
"Edit any page with Elementor, and in the widget panel, search for 'Starter Templates'. You'll find all 43 widgets ready to drag and drop."

**ON-SCREEN TEXT:** "Works with Gutenberg AND Elementor"

---

### WRAP UP (3:45 - 4:00)

**VISUAL:** Documentation page

**NARRATOR:**
"That's the basics. For detailed guides on each block, check the documentation—link in the description."

**VISUAL:** Return to clean WordPress dashboard

**NARRATOR:**
"Thanks for watching. If you have questions, leave a comment or contact support."

**ON-SCREEN TEXT:** "Documentation: docs.wbcomdesigns.com"

---

## Production Notes

### Recording Checklist

Before recording:
- [ ] Clean WordPress installation
- [ ] BuddyPress installed and activated (to show BP blocks)
- [ ] Sample content that looks realistic
- [ ] Browser zoomed appropriately (125% usually works)
- [ ] No personal data visible
- [ ] Notifications disabled

### Annotation Suggestions

| Timestamp | Annotation |
|-----------|------------|
| 0:30 | Circle around download button |
| 1:00 | Arrow pointing to Upload Plugin |
| 1:20 | Highlight Activate button |
| 1:45 | Circle + button in editor |
| 2:00 | Box around search field |
| 2:45 | Arrow pointing to Theme Colors toggle |

### Common Questions to Address

Consider adding FAQ chapter at end:
- "Why don't I see BuddyPress blocks?" - BuddyPress must be active
- "Blocks don't appear after install" - Try refreshing the page
- "Colors look wrong" - Enable Theme Colors toggle

### Accessibility Notes

- Speak clearly and at moderate pace
- Describe visual actions verbally
- Use high contrast for any annotations
- Add captions/subtitles
