# Installation & Setup Tutorial Video Script

## Video Details

- **Title:** How to Install Wbcom Essential (2-Minute Setup)
- **Duration:** 2:00 - 2:30 minutes
- **Purpose:** Walk users through installation and first widget
- **Tone:** Helpful, clear, step-by-step
- **Target:** New users, non-technical users

---

## Script

### INTRO (0:00 - 0:10)

**VISUAL:** Wbcom Essential logo, then WordPress dashboard

**NARRATOR:**
"In this quick tutorial, I'll show you how to install Wbcom Essential and add your first widget. It takes about 2 minutes."

**ON-SCREEN TEXT:** "2-Minute Setup Guide"

---

### REQUIREMENTS (0:10 - 0:20)

**VISUAL:** Theme appearance page showing BuddyX active

**NARRATOR:**
"First, make sure you have BuddyX, BuddyX Pro, or Reign theme active. You'll also need either Elementor or the built-in block editor — both work great."

**ON-SCREEN TEXT:**
- "✓ BuddyX / BuddyX Pro / Reign theme"
- "✓ Elementor or Block Editor"

---

**VISUAL:** Screen recording of installation process

**NARRATOR:**
"The easiest way to install is from your WordPress dashboard."

**VISUAL:** Navigate to Plugins → Add New

**NARRATOR:**
"Go to Plugins, then Add New."

**ON-SCREEN TEXT:** "Plugins → Add New"

---

**Step 2**
**VISUAL:** Search "wbcom essential"

**NARRATOR:**
"Search for 'Wbcom Essential'."

**ON-SCREEN TEXT:** Search box with "wbcom essential"

---

**Step 3**
**VISUAL:** Click Install Now, then Activate

**NARRATOR:**
"Click Install Now, then Activate. That's it — you're ready to go."

**ON-SCREEN TEXT:** "Install → Activate"

### METHOD 2: UPLOAD (1:00 - 1:20)

**VISUAL:** Alternative upload method

**NARRATOR:**

**ON-SCREEN TEXT:** "Plugins → Add New → Upload Plugin"

**VISUAL:** Elementor editor opening

**NARRATOR:**

**NARRATOR:**

**ON-SCREEN TEXT:** "Wbcom Essential category"

## Production Notes

### Visual Style
- Clear, visible cursor movements
- Highlight/zoom on important elements
- Clean WordPress dashboard (hide unrelated plugins)
- Slow enough to follow, fast enough to stay engaging

### Music
- Soft, unobtrusive background
- Tutorial/educational feel
- Lower volume to prioritize narration

### Screenshots/Recordings Needed
1. WordPress dashboard with BuddyX theme active
2. Plugins → Add New page
3. Search results for "wbcom essential"
4. Install and Activate process
5. Upload plugin flow (alternative)
6. Elementor editor with widget panel
7. Block editor with /wbcom search
8. Members Grid widget drag and customize
9. Publish button and preview

### Accessibility
- Clear narration
- On-screen text for all key steps
- Good contrast in recordings
- Captions available
