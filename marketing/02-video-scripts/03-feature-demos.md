## Demo 1: Theme Colors Feature

**Duration:** 2-3 minutes
**Audience:** All users

### Script

**[Opening shot: WordPress editor with a block]**

"One of the most frustrating parts of using blocks is matching colors to your theme. You add a testimonial carousel, and suddenly you're picking primary colors, secondary colors, background colors..."

**[Show manual color picking - fast forward]**

"With WBcom Essential, there's a better way."

**[Click Theme Colors toggle]**

"Every block has a 'Use Theme Colors' toggle. Enable it, and the block automatically inherits your theme's color scheme."

**[Show color change happening]**

"No manual picking. No guessing hex codes. The block reads CSS custom properties from your theme and applies them automatically."

**[Show same block on different themes]**

"This works especially well with Theme Reign, BuddyX, and BuddyX Pro - but any theme that defines CSS variables will work."

**[Closing]**

"Theme Colors. One toggle. Automatic styling. Find it in any Starter Pack block."
