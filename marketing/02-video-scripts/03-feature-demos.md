## Production Notes (All Demos)

### Standard Elements
- 5-second branded intro
- Consistent cursor style
- Same music library
- Branded lower thirds
- End card with links

### Recording Tips
- Use demo content that looks realistic
- Hide irrelevant Elementor panels
- Show responsive previews when relevant
- Keep mouse movements smooth

### Call to Action (All Videos)
End with:
- Subscribe prompt
- Link to Wbcom Essential download
- Link to related demos

### Thumbnails
- Widget name prominently displayed
- Screenshot of widget in action
- Consistent brand colors
- "FREE" badge
