### Short Descriptions

**One-liner:**
> 81 free widgets and blocks for BuddyX, BuddyX Pro, and Reign themes.

**Elevator pitch (30 words):**
> Wbcom Essential is the free companion plugin for BuddyX and Reign themes. 43 Elementor widgets, 38 Gutenberg blocks, 11 BuddyPress-specific — all designed by the theme team.

**Extended description (100 words):**
> Wbcom Essential unlocks the full potential of your BuddyX, BuddyX Pro, or Reign theme with 81 free widgets and blocks. Built by the same team that created your theme, it includes 43 Elementor widgets and 38 Gutenberg blocks — 11 of which are designed specifically for BuddyPress communities. Display members in beautiful grids, showcase groups with carousels, surface forum activity, and build trust with testimonial displays. All without paying for premium addons or writing custom code. Why is it free? Because when your site looks great, our themes look great — and that's good for everyone.

### WordPress.org Description

```
Wbcom Essential is the free companion plugin for BuddyX, BuddyX Pro, and Reign themes.

**81 widgets and blocks** designed specifically for your theme — built by the same team.

= What's Included =

* **43 Elementor Widgets** — Drag-and-drop visual editing
* **38 Gutenberg Blocks** — Native block editor support
* **11 BuddyPress Widgets** — Member grids, group carousels, forum feeds
* **5 WooCommerce Widgets** — Product displays and banners

= Why Free? =

We build BuddyX, BuddyX Pro, and Reign themes. This plugin makes YOUR site look better — which makes our themes more valuable. When you win, we win.

= Key Features =

* Perfect theme compatibility (same dev team)
* Assets only load when widgets are used
* WordPress coding standards compliant
* Regular updates with theme releases
* No premium version, no upsells

= Requirements =

* WordPress 6.0+
* PHP 8.0+
* BuddyX, BuddyX Pro, or Reign theme
* Elementor (optional) or Block Editor
```
