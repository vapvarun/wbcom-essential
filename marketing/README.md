## Product Summary

**Product:** WBcom Essential
**Version:** 4.2.0
**Tagline:** Build stunning WordPress sites with blocks and widgets that actually work together.

**What It Is:**
A companion plugin providing 45 Gutenberg blocks and 43 Elementor widgets for BuddyPress communities, WooCommerce stores, and content-rich websites.

**Key Numbers:**
- 45 Gutenberg blocks
- 43 Elementor widgets
- 11 BuddyPress integrations
- 7 bonus blocks (Counter, CTA Box, Divider, Icon Box, Mini Cart, Social Icons, Star Rating)
- Works with any theme (optimized for Theme Reign, BuddyX, and BuddyX Pro)
- Completely free download

## Brand Voice Quick Reference

**Do:**
- Write like a helpful colleague
- Lead with benefits, not features
- Use "so you can..." to connect features to outcomes
- Be specific about what things do

**Don't:**
- Use buzzwords (revolutionary, game-changing, seamless)
- Overuse emojis (max 1 per piece)
- Sound like a marketing robot
- Make vague claims without specifics
