- B) WBcom Essential + WordPress [X.X] - all good
- C) Compatibility update ready

---

**Email Body:**

Hey [Name],

WordPress [X.X] is out, and WBcom Essential v[X.X.X] is fully compatible.

**What we tested:**
- All 45 blocks work in the new editor
- Theme Colors feature functions correctly
- BuddyPress integration remains stable
- No conflicts with new WordPress features

**Should you update?**
If you're updating WordPress to [X.X], update WBcom Essential first. This ensures everything works smoothly.

**Update order:**
1. Back up your site
2. Update WBcom Essential (Dashboard → Updates)
3. Update WordPress

If you've already updated WordPress and things are working, you're probably fine—but updating WBcom Essential ensures you have the latest fixes.

Questions? Reply to this email.
