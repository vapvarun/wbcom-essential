## FAQ Section

## Frequently Asked Questions

**Is Wbcom Essential really free?**
Yes, 100% free. No premium version, no upsells, no feature limitations.

**Does it work with my theme?**
Wbcom Essential is built for BuddyX, BuddyX Pro, and Reign themes. It may work with other themes but isn't officially supported.

**Do I need Elementor Pro?**
No. Wbcom Essential works with the free version of Elementor. You also get Gutenberg blocks if you don't use Elementor at all.

**Will it slow down my site?**
No. Widgets only load assets when used. The plugin is optimized for performance.

**Is it compatible with the latest WordPress?**
Yes, tested with WordPress 6.9 and BuddyPress 14.x.

**Where do I get support?**
Documentation is available at developer.wbcomdesigns.com. For bugs, use our support forum.
