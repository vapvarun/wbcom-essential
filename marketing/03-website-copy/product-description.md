## Full Description (250 words)

**Build Professional WordPress Sites with 45 Blocks and 43 Widgets**

WBcom Essential is a companion plugin for WordPress sites, providing production-ready Gutenberg blocks and Elementor widgets.

**What's Included:**
- 45 Gutenberg blocks across 7 categories
- 43 Elementor widgets for drag-and-drop editing
- 11 BuddyPress integrations (Members Grid, Groups Carousel, Profile Completion, and more)
- 7 bonus blocks (Counter, CTA Box, Divider, Icon Box, Mini Cart, Social Icons, Star Rating)

**Key Features:**

*Theme Colors Toggle*
Enable "Use Theme Colors" and blocks inherit your theme's color palette. No manual color picking—everything matches automatically.

*BuddyPress Ready*
Built by the team behind Theme Reign and BuddyX themes. Display members in grids or carousels, show group directories, and add profile completion widgets that encourage user engagement.

*Dual Editor Support*
Use Gutenberg blocks or Elementor widgets—your choice. Same features available in both editors.

*Performance Focused*
Conditional loading ensures only active blocks load on each page. Modern Swiper v11 carousels run smoothly on any device.

**Block Categories:**
- Header: Branding, Smart Menu, Header Bar, Login Form
- Design: Shapes, Dividers, Icons, Progress Bars
- Content: Tabs, Accordions, Flip Boxes, Testimonials
- Blog: Post Carousels, Sliders, Timelines, Tickers
- Marketing: Pricing Tables, CTAs, Countdowns, Counters
- BuddyPress: Members, Groups, Forums, Activity
- WooCommerce: Product Grid, Mini Cart

Works with any properly coded WordPress theme. Optimized for Theme Reign, BuddyX, and BuddyX Pro.

## WordPress.org Description

**WBcom Essential - Gutenberg Blocks & Elementor Widgets**

Build professional WordPress sites without code. WBcom Essential provides 45 Gutenberg blocks and 43 Elementor widgets, including 11 dedicated BuddyPress integrations.

**Features:**

= Theme Colors =
Toggle "Use Theme Colors" and blocks match your theme automatically. Works with themes using CSS custom properties.

= BuddyPress Integration =
11 blocks built specifically for community sites:
* Members Grid - Display members in customizable grid
* Members Carousel - Smooth member slider
* Groups Grid - Show BuddyPress groups
* Group Carousel - Groups in carousel format
* Profile Completion - Progress indicator for user profiles
* Forums - Display bbPress forums
* Forums Activity - Recent forum posts
* Dashboard Intro - Welcome section for members
* Members Lists - Alternative list layout
* Groups Lists - Groups in list format
* Header Bar - Community navigation

= Block Categories =
* Starter Pack - Header (4 blocks)
* Starter Pack - Design (14 blocks)
* Starter Pack - Content (8 blocks)
* Starter Pack - Blog (8 blocks)
* Starter Pack - Marketing (4 blocks)
* Starter Pack - BuddyPress (11 blocks)
* Starter Pack - WooCommerce (2 blocks)

= Bonus Blocks =
7 blocks exclusive to Gutenberg:
* Counter - Animated number counter
* CTA Box - Call-to-action section
* Divider - Decorative separators
* Icon Box - Icon with title and text
* Mini Cart - WooCommerce cart widget
* Social Icons - Social media links
* Star Rating - Review rating display

= Requirements =
* WordPress 6.0+
* PHP 8.0+
* Optional: Elementor 3.0+ (for widgets)
* Optional: BuddyPress 10.0+ (for BP blocks)
* Optional: WooCommerce 7.0+ (for WC blocks)

= Documentation =
Full documentation available at [docs.wbcomdesigns.com](https://docs.wbcomdesigns.com/docs/wbcom-essential/)
